# Brand assets

All app icons and logos live here. To rebrand the app, replace these files
**with the same filenames** — no code changes required:

| File                      | Where it's used                                    | Recommended size |
| ------------------------- | -------------------------------------------------- | ---------------- |
| `logo.png`                | Navbar logo (light mode)                           | 256×256 (square) |
| `logo-dark.png`           | Navbar logo (dark mode, falls back to `logo.png`)  | 256×256 (square) |
| `favicon.ico`             | Browser tab icon                                   | 16/32/48/64      |
| `apple-touch-icon.png`    | iOS home-screen icon                               | 180×180          |
| `icon-192.png`            | PWA / Android install icon                         | 192×192          |
| `icon-512.png`            | PWA splash + high-res install icon                 | 512×512          |

Keep filenames identical and use transparent PNGs (except favicon.ico).

The current set is an AI-generated placeholder. Replace whenever a final brand
asset is available.

App name, theme color and PWA metadata are configured in `public/manifest.json`.
